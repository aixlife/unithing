# Print PDF Layout Rule

유니띵 인쇄/PDF 출력은 화면 캡처가 아니라 상담용 A4 문서 모드로 다룬다.

## 핵심 원칙

- 저장된 분석 내용, 원문, 점수, 추천, 순서는 삭제/요약/변경하지 않는다.
- 화면용 버튼, 탭, 네비게이션, 액션 UI만 출력에서 제외한다.
- 문서 구조는 큰 주제 → 소주제 → 학년별 큰 박스 흐름을 유지한다.
- 학년별 큰 박스 안은 학년 제목 → 원문 → 분석 요약 순서로 구성한다.
- 긴 원문은 페이지를 자연스럽게 넘어가도록 허용한다.
- 분석 요약 박스는 원문과 시각적으로 분리하고, 가능하면 한 덩어리로 유지한다.
- 큰 학년 박스 전체에 `break-inside: avoid`를 걸지 않는다. 작은 카드와 분석 요약 영역에만 필요한 만큼 적용한다.
- 인쇄용 폰트 크기, 줄간격, 여백은 A4 가독성을 기준으로 화면용 스타일과 다르게 조정한다.
